# Backend

Fastify + TypeScript API service for document ingestion and question answering with citations.

## Scripts

- `npm run dev`: start server in watch mode
- `npm run check`: type check
- `npm run lint`: run lint
- `npm run build`: compile to `dist/`
- `npm run start`: run compiled build

## Environment

Use `.env` in this folder and update values as needed.

## Current API

- `GET /health`
- `GET /documents` (placeholder)
- `POST /documents` (placeholder)
- `POST /ask` (placeholder)
